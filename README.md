# Описание проектов

Данные проекты были выполнены в рамках исследовательской деятельности в Научно-исследовательском центре предпринимательства Назарбаев Университета ([NURCE](https://nurce.kz/)).

| Название проекта | Описание | Используемые библиотеки |
| -------- | -------- | -------- |
| [Financing choices of entrepreneurs in the venture lifecycle](https://github.com/DSBmk/startup-research/tree/main/financing-choice-research) | Проект посвящён исследованию источников финансирования МСБ в Казахстане с использованием количественного и качественного анализа для изучения финансового поведения предпринимателей и факторов выбора финансовых инструментов. | `pandas`, `numpy`, `matplotlib`, `seaborn`, `SciPy`|
| [Learning mechanisms in technology business accelerators](https://github.com/DSBmk/startup-research/tree/main/accelerator-research) | Проект посвящён исследованию механизмов обучения в акселераторах стартапов с использованием смешанного подхода, качественного анализа и описательной статистики. | `pandas`, `matplotlib`, `seaborn` |
| [Исследование структуры финансирования стартапов](https://github.com/DSBmk/startup-research/tree/main/startup-financing-research) | Проект посвящён исследовательскому анализу венчурного финансирования стартапов на основе данных CB Insights с изучением структуры инвестиций, динамики раундов финансирования и эффективности различных инвестиционных стратегий. | `pandas`, `numpy`, `matplotlib`, `seaborn` |
