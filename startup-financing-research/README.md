# Исследование венчурного финансирования стартапов
## О проекте
В рамках проекта был проведён исследовательский анализ данных венчурного финансирования стартапов на основе датасетов платформы CB Insights по состоянию на начало 2015 года. Исследование направлено на изучение структуры, динамики и эффективности различных типов финансирования, а также выявление особенностей инвестиционного рынка стартапов.
## Датасеты
В исследовании использовались два датасета платформы CB Insights:
* `cb_investments.csv` — основной датасет с информацией о раундах венчурного финансирования стартапов с 1921 года по 31 декабря 2014 года; 
* `cb_returns.csv` — вспомогательный датасет с данными о суммах возвратов по типам финансирования в миллионах долларов начиная с 2000 года. 
## Цель исследования
Подготовить датасет к анализу, исследовать динамику и структуру финансирования стартапов, а также определить наиболее устойчивые и перспективные инвестиционные стратегии с позиции инвестора начала 2015 года.
## Использованные библиотеки
* `pandas` — обработка и анализ данных; 
* `numpy` — численные вычисления; 
* `matplotlib` — визуализация данных; 
* `seaborn` — статистическая визуализация.
